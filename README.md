# Ballanced

## Introduction

This is the WIP reversed source code for Ballance. The source codes in this branch are written to restore the original as much as possible. However, there are several deliberate modifications:

- All text written in German is translated into English.
- Some bugs that exist in the original are fixed.

The recompiled player and building blocks have been tested on Windows XP and newer versions of Windows.

## Installation

- Ballanced requires uncompleted DLL files and assets of the game.
- Build it yourself or download the latest build.
- Copy the build files into the game root directory.

## Building from Source

This project uses Visual Studio 6.0 as the original does.

Clone the repository with `git clone https://github.com/doyaGu/Ballanced.git`. Then `cd Ballanced` into the cloned repository. Now open `Ballance.dsw`, and compile the projects.
